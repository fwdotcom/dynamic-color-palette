What's in the Download

There is one download:

  • dynamic_color_palette.zip — Blender addon (GPL v3)
    Requires Blender 4.2 or later.

The Godot 4 shader files (dcp_multicol.gdshader, dcp_singlecol.gdshader) and the GDScript utility class (dcp_util.gd) are exported directly from the DCP Configure dialog into your Godot project directory — no separate shader package needed.

Installation — Blender Addon

  1. Download dynamic_color_palette.zip.
  2. Open Blender 4.2 or later.
  3. Drag and drop the ZIP file into the Blender window — the Extension Manager installs it automatically.
  4. The DCP tab appears in the 3D Viewport N-Panel (press N to open it), visible in both Object Mode and Edit Mode.

Setup — Godot 4 (Multicolor workflow)

  1. In the DCP Configure dialog, set the GDShader export path to a directory inside your Godot project (e.g. res://dcp/).
  2. Click Generate / Regenerate Palette. DCP writes dcp_multicol.gdshader (and dcp_singlecol.gdshader) to that directory.
  3. Select your mesh node in the Godot scene.
  4. In the Inspector, assign a new ShaderMaterial to the mesh's material slot.
  5. Set the ShaderMaterial's Shader property to dcp_multicol.gdshader.
  6. Assign the two textures exported from Blender to the shader parameters:
    • albedo_tex → dcp_albedo.png — import as sRGB, filter: Nearest, repeat: Disabled
    • material_tex → dcp_material.png — import as Linear (non-color), filter: Nearest, repeat: Disabled

Important: material_tex must be imported as non-color / linear. If Godot applies sRGB to it, PBR values will be wrong. Use Nearest filtering on both textures — bilinear filtering blurs adjacent palette cells into each other, producing incorrect colors at cell edges.

Quick Start (Blender)

  1. Open the DCP tab in the N-Panel.
  2. Click Configure… — adjust columns, rows, and PBR values, or leave the defaults.
  3. Click Generate / Regenerate Palette inside the dialog.
  4. Open an Image Editor next to the 3D Viewport. Click Pick From Image Editor in the DCP panel.
  5. Click a color cell to preview it, or select faces first and click to assign immediately.
  6. Use Assign Multicolor Material (UV-driven) or Assign Singlecolor Material (baked) as needed.
  7. Set the export paths in Configure… — DCP writes textures, JSON config, shaders, and the GDScript util to the configured directories on every Generate run.
